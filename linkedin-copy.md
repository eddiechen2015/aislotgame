## Experience Section Version

### AI-Assisted Game Engineering Showcase

Designed and implemented a production-oriented slot math engine platform from a TypeScript 243-ways cascading slot prototype.

Key contributions:

- Reworked the ways evaluator to support exact per-way wild multiplier capping using dynamic programming.
- Replaced independent weighted cell picks with deterministic reel strips, stop positions, and visible windows.
- Added separate base-game and free-spin reel sets and paytables.
- Built a dedicated settlement layer using integer cents, atomic payout-event rounding, per-spin caps, and market-specific absolute win caps.
- Introduced runtime math profiles for safe in-memory testing of alternative reel and paytable configurations.
- Built automated RTP search across base/FS paytables, scatter pays, and reel-count deltas.
- Added multi-seed verification with statistical confidence reporting, including standard error and 95% confidence intervals.
- Added approved-profile runtime gates to prevent unverified math profiles from being loaded in production-like environments.
- Implemented audit verification with RNG-trace replay to reproduce recorded rounds and compare raw, settled, cap, and wallet accounting fields.
- Added deterministic regression tests for payout math, settlement precision, profile loading, audit records, and replay behavior.
